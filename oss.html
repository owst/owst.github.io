London-based Polyglot Programmer

A list of open source contributions I've made; some are small, but it's nice to be able to remember them:

  • Clojure Elixir NodeJS PHP Python Ruby

    I fixed a bug, normalised multi-language implementations and improved test coverage of the Mailchecker project that rejects invalid/throwaway email addresses

  • Ruby

    I improved several linters of the popular Rubocop code checking tool:

    1. Avoiding nested method definition warnings
    2. Treating loops as conditionally executed to avoid potentially-incorrect useless variable assignment warnings,
    3. Ensuring colons in optional keyword arguments are followed by spaces,
    4. Removing duplicate warnings regarding spacing around general and ternary operators,
    5. Catching case statements with empty conditions,
    6. Improving useless access modifier warnings,
    7. Recognising useless array splats
  • Haskell

    I was a core member of the Darcs VCS team, contributing many patches

  • Python

    I added support for IPv6 extension headers to the dpkt library, the google code issue no longer exists, but the changelog (now on GitHub) bears my name

  • Haskell

    I improved certain error messages of the Cabal packaging tool

  • Haskell

    I added an instance to the Boolean algebra library Cond to recognise that Boolean algebras are closed under pairing
